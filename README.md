# Die Again: Troll Game Ever

Оригинальный платформер на Python/Pygame с нечестными ловушками, смертями, перезапусками и короткими комнатами.

Без чужих персонажей, сюжета, боевых меню, музыки, спрайтов и узнаваемых механик из других игр.

## Запуск

```powershell
python game.py
```

## Управление

- `A/D` или стрелки: движение
- `W`, `Up` или `Space`: прыжок
- `R`: перезапуск комнаты
- `Enter`: начать заново после концовки
- `Esc`: выход

## Что уже есть

- двенадцать комнат с ловушками
- Фальшивые двери
- Физические силуэты скрытых шипов и платформ, которые раскрываются по продвижению игрока
- Платформы, исчезающие после касания
- Счётчик смертей
- Процедурные пиксельные текстуры для пола, стен, шипов, дверей и героя
